## Compilaton Commands
### Build and Run C++
```
cmake -S . -B build -DPROBLEM_FILE=main.cpp
cmake --build build
cmake --build build --target run 
```

### Build and Run Rust
```
cargo build
cargo run < input.txt | tee output.txt
```
